# 📚 lesson-prep — 小学全科备课 Skill v3.0

> **史诗级增强版** — 整合人教版《教师教学用书》+ 昌明师友《教学设计与指导》详案，生成专业级教案、TOP设计师级PPT提示词、智能分层练习题。

## ✨ 核心特性

### 📖 详案级教案（非简案）
- 深度整合**两大核心教参**的完整详案内容
- 教学过程包含**逐句话术**、**预设与生成**、**课堂调控策略**
- 每个环节精确到分钟，师生活动双栏对照
- 教材分析1000-2000字，学情分析400-600字
- 输出：专业 `.docx` Word文档

### 🎨 TOP专业设计师级PPT提示词
- 完整**设计系统**（Design System）：配色/字体/栅格/动效
- 每页含**15+维度**：栅格布局、文字内容（含字号字重）、配图详细要求、动画时间线、教师备注、可访问性
- 科目专属视觉语言：语文→新中式水墨风 / 数学→极简几何风 / 英语→活泼国际化风 / 科学→自然探索风
- 配图资源库 + 搜索关键词 + 备选方案
- 输出：设计师可直接执行的 `.txt` 提示词

### ✍️ 智能分层练习题
- 严格三级分层：★基础(55%) / ★★提高(30%) / ★★★拓展(15%)
- 全知识点覆盖，题型丰富
- 每题附答案 + 详细解析 + 易错提醒
- A4双面打印优化，紧凑排版
- 输出：专业 `.pdf` 文件

```
备课，六年级英语下册外研社版Module 4
```

## 🏗️ 技术架构

```
lesson-prep/
├── SKILL.md                          # Skill主文档（工作流程+规范）
├── README.md                         # 本文件
├── scripts/
│   ├── parse_input.py                # 自然语言解析（提取备课参数）
│   ├── generate_lesson_plan.py       # 详案Word文档生成器
│   ├── generate_ppt_prompt.py        # TOP级PPT提示词生成器
│   └── generate_exercises.py         # 分层练习题PDF生成器
└── templates/
    └── subjects_config.json          # 全科目配置（题型/难度/目标模板）
```

## 📊 支持范围

| 维度 | 支持内容 |
|------|----------|
| 年级 | 小学1-6年级 |
| 科目 | 语文、数学、英语、科学、道德与法治、音乐、美术、体育、信息技术 |
| 版本 | 人教版/统编版/科教版/北师大版/苏教版/外研社版 等20+版本 |

## 🔧 依赖

- Python 3.8+
- `python-docx`（Word文档生成）
- `fpdf2`（PDF生成）
- 系统中文字体（wqy-microhei 或 NotoSansCJK）

```bash
pip install python-docx fpdf2
```
